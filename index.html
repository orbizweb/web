Cloud Security Architect

Level: Senior (7+ years)  |  Focus: Governance & GCC Compliance

The Role:

In our mission to be a GCC Enabler, security is our top priority. You will be responsible for designing the "SecureGuard" framework for our global clients. You will ensure that as companies scale their operations into new territories, their data remains compliant with international standards (GDPR, SOC2, HIPAA).

Key Responsibilities:
  • Design and implement end-to-end security architectures for multi-cloud environments.
  • Automate security audits using CloudAudit AI tools to identify vulnerabilities in real-time.

  • Certifications: CISSP, CCSP, or AWS Certified Security Specialty.
  • Experience with DevSecOps pipelines and container security (Prisma Cloud/Aqua).
